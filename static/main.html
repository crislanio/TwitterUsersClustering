Here you can choose how many users you want to crawl from Time.mk Top Twitter users and insert them locally. There are 15 pages, so you can choose values from 1 to 15. The Twitter API rate limit is strict, so if you cannot dedicate your PC long, you can fill the database incrementally. One page contains 20 users.

You should call this method after updating the database. When you update the database the users aren't clustered. If you want to cluster them, run this code and the cluster numbers will be calculated and written in database.

You should call this method when you want to see the existing clusters. If the database has been updated and the clusters have not been updated, all users will be in one cluster.
